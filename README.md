
## Getting Started

### Prerequisites
- Python 3.x
- Required packages installed from `requirements.txt`

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jamil91m/Saint-Antonys-Senior-Secondary-School.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Saint-Antonys-Senior-Secondary-School
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

### Usage
- To start the application:
  ```bash
  python main.py
  ```
